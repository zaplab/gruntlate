gruntlate
=========

Basic Grunt Template

## Setup

```
$ npm install
```

npm install should call:

```
$ bower install
$ grunt setup
```


## Usage

```
$ grunt
$ grunt serve
```

### If we take use of isDevMode inside Gruntfile.js

development (default) or production output (for example css compression on/off):

```
$ grunt --target=production
$ grunt --target=prod
```
